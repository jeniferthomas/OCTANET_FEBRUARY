# To-Do List Project

This project is a simple To-Do List application built using HTML, CSS, and JavaScript. It allows users to add tasks, mark them as completed, and delete them.

## Features

- Add new tasks to the list.
- Mark tasks as completed.
- Delete tasks from the list.
- Responsive design for various screen sizes.
- Data persistence using local storage.

## Installation

1. Clone this repository to your local machine.

2. Open the project folder in your preferred text editor.

3. Open `index.html` in a web browser to use the application.

## Usage

- Type your task into the input field and press enter to add it to the list.
- Click on a task to mark it as completed.
- Click on the trash icon next to a task to delete it from the list.
- Tasks are saved to local storage, so they will persist even after refreshing the page.
